The basic principles of CSS

Box-model:

Display

block Элемент показывается как блочный. Происходит перенос строк в начале и в конце содержимого. Внутри могут содержаться как блочые(block), так и встроенные элементы(inline).
inline Элемент отображается как встроенный. Отменяет перенос строк в начале и в конце содержимого. Элементы находятся радом друг с другом. Внутри могут содержаться только inline элементы.
inline-block Это значение генерирует блочный элемент, который обтекается другими элементами веб-страницы подобно встроенному элементу. Внутри могут содержаться как блочые(block), так и встроенные элементы(inline).
list-item Элемент выводится как блочный и добавляется маркер списка.
none Временно удаляет элемент из документа. Занимаемое им место не резервируется и веб-страница формируется так, словно элемента и не было.

Position schemes

  • Standard
  • Float
  • Absolute

Standard position scheme

static Стандарное расположение элемента на странице. Его позиция завист от других элементов и использование свойств left, top, right и bottom не приводит к каким-либо результатам.
relative Под элемент резервируется место под его размеры, а положение элемента устанавливается относительно его исходного места. Добавление свойств left, top, right и bottom изменяет позицию элемента и сдвигает его в ту или иную сторону от первоначального расположения.

Float position scheme

left Выравнивает элемент по левому краю, а все остальные элементы обтекают его по правой стороне.
right Выравнивает элемент по правому краю, а все остальные элементы обтекают его по левой стороне.

Absolute position scheme

absolute Положение элемента задается свойствами left, top, right и bottom, также на положение влияет значение свойства position родительского элемента. Так, если у родителя значение position установлено как static или родителя нет, то отсчет координат ведется от края окна браузера. Если у родителя значение position задано как fixed, relative или absolute, то отсчет координат ведется от края родительского элемента.
fixed По своему действию это значение близко к absolute, но в отличие от него привязывается к указанной свойствами left, top, right и bottom точке на экране и не меняет своего положения при прокрутке веб-страницы.

Selectors

Basic selectors

tagname Селектор по тегу.
#elementId Селектор по уникальному ID элемента. Задается с помощью аттрибута id.
.className Селктор по классу. В атрибуте class элемента могут указываться несколько значений через пробел. Классы могут повторяться у разных элементов.
* Выбирает все элементы.
[attrName] Селектор по аттрибуту. Найдет все элементы у которых присутствует данный аттрибут либо содержит нейкое определенное значение.
selector:pseudoClass Применяется если элемент находиться в определенном состоянии. Состояние определяется с помощью псевдокласса.
selector:pseudoElement Позволяет задать стиль для определенного места в элементе. Например для первой буквы или строки.

Combinations of selectors

selector1 selector2 Контекстно-зависимый селектор. Ищет все элементы удовлетворяющие selector2 и находящиеся внутри selector1.
selector1selector2 Выбор элементов, одновременно удовлетворяющие и selector1 и selector2. Можно трактовать как условие с И.
selector1, selector2 Выбор элементов, удовлетворяющих или selector1 или selector2. Можно трактовать как условие с ИЛИ.
selector1 + selector2 Стиль при такой записи применяется к selector2, но только в том случае, если он является братом для selector1 и следует сразу после него.
selector1 > selector2 Стиль применяется к selector2, но только в том случае, если он является дочерним для selector1.

Style Inheritance

div {
  font-size: 15px; // Шрифт поменяется для всего текста, который находиться внутри DIV
  color: blue; // Шрифт станет синим
  border: 1px solid blue; // Но граница станет голубой только для самого DIV!
}
.font-red {
  color: red; // Переопределили правило!
}
